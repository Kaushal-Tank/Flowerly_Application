# 🌸 Flowerly Shopping App

🚧 **Project Status: In Progress (40–50% complete)**

An Android e-commerce application for browsing and purchasing flowers for special occasions.  
This project is currently under development and aims to implement modern Android architecture and clean UI design.

---

# 🚀 Current Features

- 🔐 User Authentication (Login & Signup)
- 🌼 Basic product listing UI
- 📱 Responsive layouts using XML
- 🧱 MVVM architecture setup

---

# 🛠 Tech Stack

- Kotlin  
- XML (ConstraintLayout)  
- Firebase Authentication  
- MVVM Architecture
